Voice — STT & TTS Pro

Full voice pipeline: upload audio, get a transcript and AI reply. Or convert any text to speech.

  • Speech-to-Text — upload audio, get transcript + AI response in one call
  • Text-to-Speech — pass text, receive audio back as base64
  • Voice Prompt — speak your prompt, stream the reply
  • Supports MP3, WAV, OGG, FLAC, WebM, M4A
# Voice prompt curl -X POST /api/voice/prompt \ -H "X-User-Id: your-user-id" \ -F "audio=@voice.mp3" \ -F '{"stream":false}'

Built for Developers

# Send a prompt curl -X POST https://jaika.url/api/prompt \ -H "Content-Type: application/json" \ -H "X-User-Id: your-user-id" \ -d '{"prompt": "Explain quantum computing"}'
# Python import requests r = requests.post("https://jaika.url/api/prompt", headers={"X-User-Id": "your-id"}, json={"prompt": "Write a haiku about code"}) print(r.json()["text"])

_persona — White-label AI

Create a skill named _persona to completely replace the default AI identity for that user.

# _persona.md You are Aria, a friendly customer support agent for Acme Corp. You only answer questions about Acme products. You always respond in a warm, professional tone. Never reveal technical details about the underlying system.
  • Each user or tenant gets a completely different AI personality
  • Other skills add to the prompt — _persona replaces it
